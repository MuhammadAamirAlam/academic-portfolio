# Muhammad Aamir Alam — Academic Portfolio

A premium, responsive academic portfolio designed to showcase professional software engineering projects, verified certifications, and academic achievements.

## Tech Stack
* **Frontend Library:** React (v18+)
* **Animation & Transitions:** GSAP (GreenSock Animation Platform)
* **Styling & Theme:** Vanilla CSS (Curated Modern Dark Theme)
* **Build Tool:** Webpack / React Scripts
* **Pre-rendering:** React Snap (Static Site Generation for SEO & performance optimization)
